剑指面试题18-java-删除列表中重复节点

Posted on 2019-12-17

题目描述:
在一个排序的链表中，存在重复的结点，请删除该链表中重复的结点，重复的结点不保留，返回链表头指针。 例如，链表1->2->3->3->4->4->5 处理后为 1->2->5

思路：
1.如果头节点是重复节点，则从第一个不重复节点开始
2.如果头节点不是重复节点，则头节点指向下一个不重复的节点，返回头节点。

代码：
/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}
*/
public class Solution {
public ListNode deleteDuplication(ListNode pHead)
{
//健壮性判断：如果列表为空或者只有一个节点，则返回该列表
if(pHead == null||pHead.next == null){
return pHead;
}
ListNode right = pHead.next;
if(right!=null&&right.val == pHead.val){
//1.如果头节点是重复节点，则从第一个不重复节点开始
while(right!=null&&right.val == pHead.val){
right = right.next;
}
return deleteDuplication(right);
}else{
//2.如果头节点不是重复节点，则头节点指向下一个不重复的节点，返回头节点。
pHead.next = deleteDuplication(right);
return pHead;
}
}
}

public boolean check(int threshold,int rows,int cols,int row,int col,boolean visited[]){
    if(row>=0&&row<rows&&col>=0&&col<cols&&getDigitSum(row)+getDigitSum(col)<=threshold&&!visited[row*cols+col]){
        return true;
    }
    return false;
}
public int getDigitSum(int num){
    int sum = 0;
    while(num > 0){
        sum += num %10;
        num/=10;
    }
    return sum;
}

}

剑指面试题12-java-矩阵中的路径

Posted on 2019-08-23

题目：给定一个矩阵，给定一组字符串，判断矩阵中是否存在一条包含字符串所有字符的路径。路径可以从矩阵中的任意一个格子开始，每一步可以再矩阵中向左向右向上向下移动，但不能两次经过格子。

思路：
①因为不能两次进入一个格子，所以需要一个和原始矩阵同维度的flag矩阵标识该格子是否可以进入。
boolean[] flag = new boolean[matrix.length];
for(int i = 0;i < matrix.length;i++){
flag[i] = true; //初始值全为true
}

②首先遍历矩阵，直到在矩阵中找到与字符串第一个值匹配的值，若找不到，则不存在路径满足。

③匹配到第i个字符所在的位置，寻找其上下左右，是否有一个位置的字符与第i+1个字符匹配

具体过程：
i）先匹配(上)：是否匹配，匹配，该位置置为false，重复③直到匹配完字符串中的所有字符；否则，该位置重新置为true，匹配(下);

ii）若匹配完上下左右，均无法匹配，则不存在一条路径满足要求，return false。

④注意越界判断，不可进入判断，不匹配判断。

我们可以看到i是一个自己调用自己的过程，独立使用一个函数，使用递归处理。

代码：
public class Solution {
public boolean hasPath(char[] matrix, int rows, int cols, char[] str)
{
//建立一个flag矩阵
boolean[] flag = new boolean[matrix.length];
for(int i = 0;i < matrix.length;i++){
flag[i] = true;
}
//寻找第一个匹配字符的位置
for(int i = 0;i < rows;i++){
for(int j = 0;j < cols;j++){
if(isKeXing(matrix, rows, cols,i,j,str,0,flag)){
return true;
}
}
}
return false;
}
//递归判断路径是否可行
public boolean isKeXing(char[] matrix, int rows, int cols,int i,int j, char[] str,int index,boolean[] flag){
//对应矩阵中的位置
int ind = i*cols+j;
//判断对应位置是否越界，是否与需要匹配的字符不相等，是否不能进入
if(i<0||i>=rows||j<0||j>=cols||matrix[ind]!=str[index]||flag[ind]==false){
return false;
}

//如果通过上述if，则将该位置置为false
flag[ind] = false;
//如果已经匹配完最后一个值了，则存在该路径
if(index == str.length-1){
    return true;
}
//递归判断是否存在该路径
if(isKeXing(matrix, rows, cols,i-1,j,str,index+1,flag)||isKeXing(matrix, rows, cols,i+1,j,str,index+1,flag)||isKeXing(matrix, rows, cols,i,j-1,str,index+1,flag)||isKeXing(matrix, rows, cols,i,j+1,str,index+1,flag)){
    return true;
}
//若上下左右都不可行
flag[ind] = true;
return false;

}
}

剑指面试题10-java-斐波那契数列

题目：现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。n<=39

递归很慢

解题思路：避免重复计算，把已经计算过的中间项保存下来。

具体代码：
public class Solution {
public int Fibonacci(int n) {
int f0 = 0;
int f1 = 1;
int f = 0;
if(n == 0){
return 0;
}else if(n == 1){
return 1;
}else{
for(int i = 2;i <=n;i++){
f = f0+f1;
f0 = f1;
f1 = f;
}
return f;
}

}

}

剑指面试题7(java):重建二叉树

Posted on 2019-06-25

题目：
输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

分析：
前序遍历序列{1,2,4,7,3,5,6,8}
中序遍历序列{4,7,2,1,5,3,8,6}

1.根节点1，在中序序列中找到1，左边为左子树，右边为右子树
2.在前序序列中找到左子树的序号范围，在中序序列中找到左子树序号范围，递归
3.在前序序列中找到右子树的序号范围，在中序序列中找到右子树序号范围，递归

代码：
/**

  • Definition for binary tree
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    public class Solution {
    public TreeNode reConstructBinaryTree(int [] pre,int [] in) {

    //健壮性判断
if(pre.length!=in.length||pre.length <= 0||in.length <= 0){
    return null;
}
TreeNode node = reConstructBinaryTree(pre,0,pre.length-1,in,0,in.length-1);
return node;

    }
    public TreeNode reConstructBinaryTree(int [] pre,int preStart,int preEnd,int [] in,int inStart,int inEnd){

       //根节点
TreeNode root = new TreeNode(pre[preStart]); 
    //在中序遍历中找到根节点，左边为左子树，右边为右子树
int leftEnd = 0;
for(int i = inStart;i <= inEnd;i++){
    if(in[i] == pre[preStart]){
        break;
    }
    leftEnd++;
}
//左子树
if(leftEnd > 0){
    //在前序遍历中确定左子树元素的序号
    int preLeftStart = preStart+1;
    int preLeftEnd = preStart+leftEnd;
    //在中序遍历中确定左子树的序号
    int inLeftStart = inStart;
    int inLeftEnd = inStart+leftEnd-1;
    //递归
    root.left = reConstructBinaryTree(pre,preLeftStart,preLeftEnd,in,inLeftStart,inLeftEnd);
}
//右子树
if((preEnd-preStart)-leftEnd > 0){
    //在前序遍历中确定右子树元素的序号
    int preRightStart = preStart+leftEnd+1;
    int preRightEnd = preEnd;
    //在中序遍历中确定右子树的序号
    int inRigthStart = inStart+leftEnd+1;
    int inRightEnd = inEnd;
     //递归
    root.right = reConstructBinaryTree(pre,preRightStart,preRightEnd,in,inRigthStart,inRightEnd);

}
return root;

    }
    }

剑指面试题6(java)：从尾到头打印链表

Posted on 2019-06-25

题目描述：
输入一个链表，按链表值从尾到头的顺序返回一个ArrayList

分析：
①ArrayList是实现了基于动态数组的数据结构，LinkedList是基于链表的数据结构
②对于随机访问get和set，ArrayList优于LinkedList，因为LinkedList要移动指针
③对于新增和删除操作add和remove，LinkedList比较占优势，因为ArrayList要移动数据。

所以显然，遍历链表，使用LinkedList保存链表中的值（addFirst），保存完后不断remove，就是从尾到头的顺序。按照书本是打印，而这里是保存到ArrayList中。


/**
  • public class ListNode {
  • int val;
  • ListNode next = null;
    *
  • ListNode(int val) {
  • this.val = val;
  • }
  • }
    /
    import java.util.LinkedList;
    import java.util.ArrayList;
    public class Solution {
    public ArrayList printListFromTailToHead(ListNode listNode) {
    LinkedList ll = new LinkedList();
    while(listNode != null){
    ll.addFirst(listNode.val);
listNode = listNode.next;
    }
    ArrayList al = new ArrayList();
    while(!ll.isEmpty()){
    al.add(ll.remove());
    }
    return al;
    }
    }

剑指面试题5：替换空格

Posted on 2019-06-25

方法一：

分析：
①空格为一个char，替换成%20为三个char，会造成覆盖，所以不能直接替换。
②非常直观的想法就是，从后面往前替换，需要对原始的字符串缓冲区进行手动扩容。
③先一次遍历，获取字符串中空格的个数，扩容容量为2*空格个数。
④设两个指针，p1指向原始字符串末尾，p2指向扩容容量末尾；
若遇到空格，则依次替换为’0’,’2’.’%’,每次p2–，最后p1–；
否则，将p2处赋值为p1处的值；p2–，p1–；
直到p1到达字符串头部，结束。

public class Solution {
public String replaceSpace(StringBuffer str) {
//遍历字符串，找到字符串中空格的个数
int len = str.length();
int count = 0;
for(int i = 0;i < len;i++){
if(str.charAt(i)==’ ‘){
count += 2;
}
}
int p2 = len+count-1;
//手动扩容
for(int i = len;i<=p2;i++){
str.append(‘0’);
}
for(int p1 = len-1;p1>=0;p1–){
if(str.charAt(p1)==’ ‘){
str.setCharAt(p2,’0’);
p2–;
str.setCharAt(p2,’2’);
p2–;
str.setCharAt(p2,’%’);
p2–;
}else{
str.setCharAt(p2,str.charAt(p1));
p2–;
}
}
return str.toString();
}
}

方法二：

分析：
使用java中StringBuffer类中的insert()函数
从前往后遍历，遇到空格则删除空格，插入’%20’
具体实现实际是由类封装的函数实现的，并没有实现时间复杂度和空间复杂度的优化，并且不利于转换为其他语言。

public class Solution {
public String replaceSpace(StringBuffer str) {
//健壮性判断
if(str == null){
return null;
}
int len = str.length();
int i = 0;
while(i<len){
if(str.charAt(i)==’ ‘){
str.deleteCharAt(i);
str.insert(i,’%’);
i++;
str.insert(i,’2’);
i++;
str.insert(i,’0’);
len = len + 2;

        }
            i++;
    }
    return str.toString();
}

}

剑指面试题4：二维数组中的查找

Posted on 2019-06-24

分析：

①若target比某个数小，则target位于该数的左侧或上方；否则位于右侧或下方。
②从右上角开始，    若target等于该数，则true；
                若target大于该数，则位于该数下方；
                若target小于该数，则位于该数左侧，row-1；
③边界条件，        要注意查找范围是否为空：array.length>0
                遍历到左上角或者左下角或者右下角。所以是row=array.length-1,col=0;


public class Solution {
public boolean Find(int target, int [][] array) {
if (array.length<=0){
    return false;
}
//右上角的点
int row = 0;
int col = array[0].length-1;
while(row < array.length && col >= 0){
    if(target == array[row][col]){
        return true;
    }
    if(target < array[row][col]){
        col--;
    }
    else{
        row++;
    }
}
    return false;
}

}

leetcode_235

Posted on 2019-03-14

题目来源：https://leetcode.com/problems/lowest-common-ancestor-of-a-binary-search-tree/

思路： 因为左子树的值>根节点>右子树，
所以当p，q的值都大于根节点的时候，说明两个点都位于根节点的右边，此时根节点不是最小共同祖先，更新根节点为原根节点的右子树；当两个值都小于根节点时，两个点都位于根节点左边，也不是最小共同祖先，更新根节点为原来根节点的左子树；直到p，q两值不同时大于或小于根节点，此时根节点为最小共同祖先

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    int parentVal = root.val;
int pVal = p.val;
int qVal = q.val;
if(pVal>parentVal&&qVal>parentVal){
    return lowestCommonAncestor(root.right,p,q);
}else if(pVal<parentVal&&qVal<parentVal){
    return lowestCommonAncestor(root.left,p,q);
}else{
    return root;
}
    }
    }
