leetcode_119

Posted on 2019-03-12

题目来源：https://leetcode.com/problems/pascals-triangle-ii/

这道题目有点意思，118题是要得到所有行，而这道题是输出给定行。也就是说，这个变化我们直接在ArrayList里面做就行，不需要全部存储起来。

规律是：
1
11
121
1331
14641
…
首先，前两行需要给定，也就是
if(rowIndex >= 0) result.add(1);
if(rowIndex >= 1) result.add(1);

而每一行最后一个元素也是1，这在确定了别的元素之后加上，就可以了。

很显然我们会用到
result.get(j)+result.get(j-1);//具体j指代什么再说。

第三行：
2=result.get(1)+result.get(0);

第四行：
第一个3 = result.get(1)+result.get(0);
这个时候result存的数据就变成了：[1,3,1]
这个时候我们就得不到第二个3了，所以显然，这个变化要从后往前做：
第二个3 = result.get(2)+result.get(1);
第一个3 = result.get(1)+result.get(0);

而我们需要的是从第二行开始一行一行变化，所以设两个循环

for(int i = 2，i <= rowIndex;i++){
//从第二行开始
for(int j = i-1,j>0;j–){
//从后往前
result.set(j,result.get(j)+result.get(j-1));
}
//最后一个元素为1
result.add(1);
}

完整代码：
class Solution {
public List getRow(int rowIndex) {
List result = new ArrayList<>();
if(rowIndex >= 0){
result.add(1);
}
if(rowIndex >= 1){
result.add(1);
}
if(rowIndex >= 2){
for(int i = 2;i <rowIndex+1;i++){
for(int j = i-1;j > 0;j–){
result.set(j,result.get(j)+result.get(j-1));
}
result.add(1);
}
}
return result;
}
}

leetcode_118

Posted on 2019-03-11

题目来源：https://leetcode.com/problems/pascals-triangle/

思路：动态规划

代码：
class Solution {
public List<List> generate(int numRows) {
List<List> list = new ArrayList<List>();
//若只有零行
if(numRows == 0){
return list;
}
//第一行元素总为1
list.add(new ArrayList<>());
list.get(0).add(1);
for(int rowNum = 1;rowNum<numRows;rowNum++){
List row = new ArrayList<>();
List prevRow = list.get(rowNum-1);
//每行第一个元素都为1
row.add(1);
for(int j = 1;j < rowNum;j++){
row.add(prevRow.get(j-1)+prevRow.get(j));
}
//每行最后一个元素为1
row.add(1);
list.add(row);
}
return list;
}
}

leetcode_113

Posted on 2019-03-11

题目来源：https://leetcode.com/problems/path-sum-ii/

代码：

/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {
    public List<List> pathSum(TreeNode root, int sum) {

    List<List<Integer>> list = new LinkedList<>();
if(root == null){
    return list;
}
List<Integer> currentList = new ArrayList<>();
currentList.add(root.val);
traverse(list,currentList,root,sum);
return list;

    }
    public void traverse(List<List> list,List currentList,TreeNode root,int sum){ 

    if(root.left == null && root.right == null){
    if (root.val == sum){
        list.add(new ArrayList<>(currentList));
    }
    return;
}
if(root.left != null){
    currentList.add(root.left.val);
    traverse(list,currentList,root.left,sum-root.val);
    currentList.remove(currentList.size()-1);
}
if(root.right != null){
    currentList.add(root.right.val);
    traverse(list,currentList,root.right,sum-root.val);
    currentList.remove(currentList.size()-1);
}

    }
    }

leetcode_112

Posted on 2019-03-11

题目来源：https://leetcode.com/problems/path-sum/

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public boolean hasPathSum(TreeNode root, int sum) {
    int s = 0;
 return dfs(root,sum,s);
    }
    public boolean dfs(TreeNode root, int sum,int s){
    if(root == null){
    return false;
}
if(root.left == null && root.right == null){
    //到达叶节点，判断和是否等于sum
    s += root.val;
    if(s == sum){
        return true;
    }
}
s += root.val;
//递归
boolean left = dfs(root.left,sum,s);
boolean right = dfs(root.right,sum,s);
if(left||right){
    return true;
}
return false;
    }
    }

leetcode_111

Posted on 2019-03-10

题目来源：https://leetcode.com/problems/minimum-depth-of-binary-tree/

思路： ①如果根节点为空，则深度为0；
②如果左子树为空，则遍历右子树深度；
③如果右子树为空，则遍历左子树深度；
④否则，遍历左右子树深度，并返回最小值；

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public int minDepth(TreeNode root) {
    if(root == null){
    return 0;
}
if(root.left == null && root.right == null){
    return 1;
}
if(root.left == null){
//注意要加上根节点的深度，即要+1
    return minDepth(root.right)+1;
}
if(root.right == null){
//同上
    return minDepth(root.left)+1;
}
return Math.min(minDepth(root.left),minDepth(root.right))+1;//同+1
    }

}

leetcode_110

Posted on 2019-03-10

题目来源：https://leetcode.com/problems/balanced-binary-tree/

思路：递归。

代码：

/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {

    boolean flag = true;

    public int dfs(TreeNode root){

    if (root == null){
    return 0;
}
if (flag == false){
    return -1;
}
int left = dfs(root.left);
int right = dfs(root.right);
if(Math.abs(left-right)>1){
    flag = false;
}
return Math.max(left,right)+1;

    }

    public boolean isBalanced(TreeNode root) {

    dfs(root);
return flag;

    }
    }

题目来源:https://leetcode.com/problems/convert-sorted-array-to-binary-search-tree/

思路：
我最开始是想从子节点往根节点做，后来发现，太困难了，好像用不了递归（因为节点命名的原因），所以还是乖乖从根节点往下做。

常规思维：找中点，二分法

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {

     if(nums.length == 0){
     return null;
 }
TreeNode head = sorted(nums,0,nums.length-1);
 return head;

    }

    public TreeNode sorted(int []nums,int low,int high){

    if(low > high){
    return null;
}
int mid = (low+high)/2;
TreeNode node = new TreeNode(nums[mid]);
node.left = sorted(nums,low,mid-1);
node.right = sorted(nums,mid+1,high);
return node;

    }
    }

leetcode_102

Posted on 2019-03-10

题目来源：https://leetcode.com/problems/binary-tree-level-order-traversal/

这道题和107题刚好是相反的，所以将每层节点的值加入最终结果的Arraylist的尾部即可

代码：

/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public List<List> levelOrder(TreeNode root) {
    Queue<TreeNode> queue = new LinkedList<>();
List<List<Integer>> result = new ArrayList<>();
queue.add(root);
while(!queue.isEmpty()){
    List<Integer> currentResult = new ArrayList<>();
    int size = queue.size();
    for(int i = 0;i < size;i++){
        TreeNode node = queue.remove();
        if (node != null){
            currentResult.add(node.val);
            queue.add(node.left);
            queue.add(node.right);
        }
    }
    if(!currentResult.isEmpty()){
        result.add(currentResult);
    }
}
return result;
    }
    }

leetcodede_107

Posted on 2019-03-10

题目来源：https://leetcode.com/problems/binary-tree-level-order-traversal-ii/

思路： 按层遍历

代码：

/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public List<List> levelOrderBottom(TreeNode root) {
    Queue<TreeNode> queue = new LinkedList<>(); //记录节点
List<List<Integer>> result = new ArrayList<>(); //记录最终结果
    //将根节点假如queue中
queue.add(root);
while (!queue.isEmpty()){
    List<Integer> currentResult = new ArrayList<>(); //记录每层所有节点的值
    int size = queue.size();  //记录该层节点总数
    for(int i = 0; i < size;i++){ //遍历该层左右节点
        TreeNode node = queue.remove(); //从队列中依次取出该层节点
        if(node != null){
            currentResult.add(node.val);  //记录当点节点的值
            //将当前节点的左右子节点加入queue中，遍历所有节点后，便得到了下一层的所有节点，因为LinkedList是链表，是有序的，所以会严格按照左右节点的顺序
            queue.add(node.left);  
            queue.add(node.right);
        }
    }
    if(!currentResult.isEmpty()){
    //将当前层的所有节点的值加入result的头部，读出时才是从子节点到根节点的顺序
        result.add(0,currentResult);
    }
}
return result;
    }
    }

leetcode_104

Posted on 2019-03-10

题目来源：https://leetcode.com/problems/maximum-depth-of-binary-tree/

方法：递归

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {
    public int maxDepth(TreeNode root) {

    if (root==null){
    return 0;
}
else{
    int max_left = maxDepth(root.left);
    int max_right = maxDepth(root.right);
    int max = Math.max(max_left,max_right)+1;
    return max;
}

    }
    }
