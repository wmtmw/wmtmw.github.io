剑指面试题38-java-字符串的排列

Posted on 2020-01-02 | Edited on 2020-01-12

题目：输入一个字符串,按字典序打印出该字符串中字符的所有排列。例如输入字符串abc,则打印出由字符a,b,c所能排列出来的所有字符串abc,acb,bac,bca,cab和cba。

代码：
import java.util.ArrayList;
import java.util.Collections;
public class Solution {
public ArrayList Permutation(String str) {
ArrayList res = new ArrayList<>();
if(str==null||str.length()==0){
return res;
}
char[] ch = str.toCharArray();
Core(res,ch,0);
Collections.sort(res);
return res;
}
public void Core(ArrayList res,char[] ch,int i){
if(i == ch.length && !res.contains(String.valueOf(ch))){
res.add(String.valueOf(ch));
return;
}
for(int k = i;k < ch.length;k++){
swap(ch,k,i);
Core(res,ch,i+1);
swap(ch,k,i);
}
}
public void swap(char[] ch,int k,int i){
char temp = ch[k];
ch[k]=ch[i];
ch[i]=temp;
}
}

剑指面试题37-java-序列化二叉树

Posted on 2020-01-02

题目：
请实现两个函数，分别用来序列化和反序列化二叉树
二叉树的序列化是指：把一棵二叉树按照某种遍历方式的结果以某种格式保存为字符串，从而使得内存中建立起来的二叉树可以持久保存。序列化可以基于先序、中序、后序、层序的二叉树遍历方式来进行修改，序列化的结果是一个字符串，序列化时通过 某种符号表示空节点（#），以 ！ 表示一个结点值的结束（value!）。
二叉树的反序列化是指：根据某种遍历顺序得到的序列化字符串结果str，重构二叉树。

思路： 1.序列化二叉树时一定要以某种形式保存空节点
2.序列化和反序列化均使用递归

代码：

/*
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
String Serialize(TreeNode root) {
if(root == null){
return “#”;
}
return root.val+”!”+Serialize(root.left)+Serialize(root.right);
}
int index = 0;
TreeNode Deserialize(String str) {
if(str==null||index == str.length()){
return null;
}
if(str.charAt(index)==’#’){
index++;
return null;
}
String cur=””;
while(str.charAt(index)!=’!’){
cur+=str.charAt(index);
index++;
}
index++;
TreeNode root = new TreeNode(Integer.valueOf(cur));
root.left = Deserialize(str);
root.right = Deserialize(str);
return root;
}
}

剑指面试题36-java-二叉搜索树与双向链表

Posted on 2020-01-02

题目：输入一棵二叉搜索树，将该二叉搜索树转换成一个排序的双向链表。要求不能创建任何新的结点，只能调整树中结点指针的指向。

思路： 1.二叉搜索树是有序的，中序遍历结果也是有序的，对二叉搜索树进行中序遍历
2.根节点左子树最大节点与根节点相连，根节点与右子树的最小节点相连。
3.递归处理

代码：
/**
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/

public class Solution{
public TreeNode Convert(TreeNode root){
if(root == null){
return null;
}
//先处理左子树，获得左子树链表的头节点
TreeNode left = Convert(root.left);
TreeNode p = left;
//如果左子树不为空，则找到左子树链表的尾节点
if(p != null){
while（p.right!=null）{
p = p.right;
}
p.right = root;
root.left = p;
}
//处理右子树，获得右子树链表的头节点
TreeNode right = Convert(root.right);
if(right!=null){
root.right = right;
right.left = root;
}
return left==null?root:left;
}
}

剑指面试题35-java-复制链表的复制

Posted on 2019-12-31 | Edited on 2020-01-02

题目：输入一个复杂链表（每个节点中有节点值，以及两个指针，一个指向下一个节点，另一个特殊指针指向任意一个节点），返回结果为复制后复杂链表的head。

思路：
1.在原始链表的每个节点N后面创建对应的N’，链表变成N→N’→…
2.复制原始节点的random指针，假设原始链表中N指向S，则N’指向S’
3.拆链表，奇数位置节点连起来为原始链表，偶数位置节点连起来是复制链表

问题：在牛客上写的时候，原始链表也需要连起来，才对。

代码：
/*
public class RandomListNode {
int label;
RandomListNode next = null;
RandomListNode random = null;

RandomListNode(int label) {
    this.label = label;
}

}
*/
public class Solution {
public RandomListNode Clone(RandomListNode pHead)
{
if(pHead == null){
return null;
}
//复制节点
RandomListNode cur = pHead;
RandomListNode next,temp,temp1;
while(cur != null){
temp = cur.next;
cur.next = new RandomListNode(cur.label);
cur.next.next=temp;
cur = cur.next.next;
}
//增加random指针
cur = pHead;
while(cur != null){
cur.next.random = cur.random==null?null:cur.random.next;
cur = cur.next.next;
}
//拆链
cur = pHead;
RandomListNode newHead = cur.next;
while(cur != null){
temp = cur.next;
temp1 = cur.next.next;
cur.next = temp1;
temp.next = temp1 == null?null:temp1.next;
cur = temp1;
}
return newHead;
}
}

剑指面试题34-java-二叉树中和为某一值的路径

Posted on 2019-12-31

题目：输入一颗二叉树的跟节点和一个整数，打印出二叉树中结点值的和为输入整数的所有路径。路径定义为从树的根结点开始往下一直到叶结点所经过的结点形成一条路径。(注意:在返回值的list中，数组长度大的数组靠前)

思路：从根节点开始遍历树，因此选择前序遍历的方式
如果当前节点是叶节点，判断该路径值之和是否等于目标整数
若不是叶节点，
若该节点有左子树，则进入左子树的递归函数
若该节点有右子树，则进入右子树的递归函数
处理完当前节点的左右子树后，弹出该节点的值

因此，需要一个元素为数组的数组，和一个元素为Integer的数组

代码：
import java.util.ArrayList;
/**
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
public ArrayList<ArrayList> FindPath(TreeNode root,int target) {
ArrayList<ArrayList> res = new ArrayList<>();
if(root == null){
return res;
}
ArrayList al = new ArrayList<>();
Core(res,al,root,target);
return res;
}
public void Core(ArrayList<ArrayList> res,ArrayList al,TreeNode root,int target){
if(root == null){
return;
}
al.add(root.val);
if(root.left == null && root.right == null && root.val == target){
res.add(new ArrayList(al));
}else{
Core(res,al,root.left,target-root.val);
Core(res,al,root.right,target-root.val);
}
al.remove(al.size()-1);
}
}

注意：往元素为数组的数组中添加元素是：res.add(new ArrayList(al));

剑指面试题33-java-二叉搜索树的后序遍历序列

Posted on 2019-12-28

题目：输入一个整数数组，判断该数组是不是某二叉搜索树的后序遍历的结果。如果是则输出Yes,否则输出No。假设输入的数组的任意两个数字都互不相同。

分析：二叉搜索树，左子树的所有节点小于根节点，右子树的所有节点大于根节点
后序遍历：先遍历左右节点，再根节点。所以序列的最后一个节点是根节点。

思路：①获得根节点，遍历序列，获得左右子树分界点（遍历序列，直到某元素>根节点，则该元素之前的节点为左子树的节点）。
②判断右子树的节点中是否有比根节点小的节点，若有，则不是二叉搜索树，提前返回false。
③接着递归判断左右子树是否为二叉搜索树。

④当出现类似  6   、  6  、  6的结构时，需要进行特殊处理。
          / \    /       \
         5   7  5         7
  在本文代码中，对应结构1，会出现left==right；对应结构2、3，会出现left>=right。此时若不进行特殊处理，会出现数组越界的问题。

代码：
public class Solution {
public boolean VerifySquenceOfBST(int [] sequence) {
//如果序列为空
if(sequence.length <1){
return false;
}
boolean result = false;
result = Core(sequence,0,sequence.length-1);
return result;

}
public boolean Core(int[] arr, int left,int right){
    //特殊情况处理，难以描述，画图理解
     if(left>=right)
        return true;
     //数组最后一个元素为根节点
    int root = arr[right];
    //判断左右子树的分割位置，原理，左子树元素都比根节点小，右子树元素都比根节点大
    int leftEnd = left;
    while(arr[leftEnd]<root){
        leftEnd++;
    }
    for(int j=leftEnd;j<right;j++){
        if(arr[j]<root){
            return false;
        }
    }
    return Core(arr,left,leftEnd-1)&&Core(arr,leftEnd,right-1);
}

}

剑指面试题32-java-从上往下打印二叉树

Posted on 2019-12-28

题目：从上往下打印出二叉树的每个节点，同层节点从左至右打印。

思路：创建一个队列，队列特点：先进先出。
打印当前节点时，把其左右子节点（若存在），按顺序排入队列最后，然后取队头元素打印，以此类推，直到队列中没有值

注意：该队列类型为节点的类型。

代码：

import java.util.ArrayList;
import java.util.LinkedList;
/**
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
public ArrayList PrintFromTopToBottom(TreeNode root) {
//数组：存要返回节点
ArrayList a = new ArrayList<>();
//数组：缓存队列
LinkedList l = new LinkedList<>();
TreeNode temp;
if (root == null){
return a;
}
l.add(root);
while(!l.isEmpty()){
//取出队头元素
temp = l.poll();
a.add(temp.val);
//若左子节点不为空，则加入队列
if(temp.left != null){
l.add(temp.left);
}
//若右子节点不为空，则加入队列
if(temp.right != null){
l.add(temp.right);
}
}
return a;
}
}

剑指面试题31-java-栈的压入,弹出序列

Posted on 2019-12-28

题目：输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否可能为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如序列1,2,3,4,5是某栈的压入顺序，序列4,5,3,2,1是该压栈序列对应的一个弹出序列，但4,3,5,1,2就不可能是该压栈序列的弹出序列。（注意：这两个序列的长度是相等的）

思路：假设入栈顺序是pushA[],出栈顺序是popA[]
建一个辅助栈s，依次将pushA压入s，每压入一个元素，循环判断栈顶元素是否和popA[]当前元素是否一致，一致则pop该栈顶元素，popA索引++，直到s的栈顶元素与popA[]元素不同，继续向s压入。直到pushA的元素压完，若popA索引长度和popA长度相同，则第二个序列为第一个序列的弹出顺序。

代码：
import java.util.Stack;

public class Solution {
public boolean IsPopOrder(int [] pushA,int [] popA) {
Stack s = new Stack<>();
int index = 0;
for(int i = 0;i < pushA.length;i++){
s.push(pushA[i]);
while(!s.isEmpty()&&(s.peek()==popA[index])){
s.pop();
index++;
}
}
return index == popA.length;
}

}

剑指面试题30-java-包含min函数的栈

Posted on 2019-12-27

题目：
定义栈的数据结构，请在该类型中实现一个能够得到栈中所含最小元素的min函数（时间复杂度应为O（1））

思路一：建一个辅助栈s2，若push的元素比s2的栈顶元素小，则压入新元素，否则将栈顶元素重新压栈

s2.peek():获取栈顶元素，但不弹出。

思路二：建一个辅助栈s2，若push的元素比s2的栈顶元素小，则压入新元素。
但pop的时候有变化。当s1要pop的元素与s2的栈顶元素相同时，s2才pop

代码一：
import java.util.Stack;

public class Solution {
Stack s1 = new Stack<>();
Stack s2 = new Stack<>();

public void push(int node) {
    s1.push(node);
    if(s2.isEmpty()){
        s2.push(node);
    }else{
        int temp = node<s2.peek()?node:s2.peek();
        s2.push(temp);
    }
}

public void pop() {
    s1.pop();
    s2.pop();
}

public int top() {
    return s1.peek();
}

public int min() {
    return s2.peek();
}

}

代码二：
import java.util.Stack;

public class Solution {
Stack s1 = new Stack<>();
Stack s2 = new Stack<>();

public void push(int node) {
    s1.push(node);
    if(s2.isEmpty()){
        s2.push(node);
    }else if(node<s2.peek()){
        s2.push(node);
    }
}

public void pop() {
    if (s1.pop()==s2.peek()){
        s2.pop();
    }
}

public int top() {
    return s1.peek();
}

public int min() {
    return s2.peek();
}

}
