leetcodede_107

Posted on 2019-03-10

题目来源：https://leetcode.com/problems/binary-tree-level-order-traversal-ii/

思路： 按层遍历

代码：

/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public List<List> levelOrderBottom(TreeNode root) {
    Queue<TreeNode> queue = new LinkedList<>(); //记录节点
List<List<Integer>> result = new ArrayList<>(); //记录最终结果
    //将根节点假如queue中
queue.add(root);
while (!queue.isEmpty()){
    List<Integer> currentResult = new ArrayList<>(); //记录每层所有节点的值
    int size = queue.size();  //记录该层节点总数
    for(int i = 0; i < size;i++){ //遍历该层左右节点
        TreeNode node = queue.remove(); //从队列中依次取出该层节点
        if(node != null){
            currentResult.add(node.val);  //记录当点节点的值
            //将当前节点的左右子节点加入queue中，遍历所有节点后，便得到了下一层的所有节点，因为LinkedList是链表，是有序的，所以会严格按照左右节点的顺序
            queue.add(node.left);  
            queue.add(node.right);
        }
    }
    if(!currentResult.isEmpty()){
    //将当前层的所有节点的值加入result的头部，读出时才是从子节点到根节点的顺序
        result.add(0,currentResult);
    }
}
return result;
    }
    }

leetcode_104

Posted on 2019-03-10

题目来源：https://leetcode.com/problems/maximum-depth-of-binary-tree/

方法：递归

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {
    public int maxDepth(TreeNode root) {

    if (root==null){
    return 0;
}
else{
    int max_left = maxDepth(root.left);
    int max_right = maxDepth(root.right);
    int max = Math.max(max_left,max_right)+1;
    return max;
}

    }
    }

leetcode_101

Posted on 2019-03-08

思路：
若左子树与右子树是镜像，需满足：
①根节点值相同
②左子树的左子树与右子树的右子树相同；左子树的右子树与右子树的左子树相同

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public boolean isSymmetric(TreeNode root) {
    return isMirror(root,root);
    }
    public boolean isMirror(TreeNode t1,TreeNode t2){
    if(t1 == null && t2 == null){
    return true;
}
if(t1 == null || t2 == null){
    return false;
}
return (t1.val == t2.val) && (isMirror(t1.left,t2.right)) && (isMirror(t1.right,t2.left));
    }
    }

leetcode_665

Posted on 2019-03-08

审题：
二叉搜索树：又称二叉排序树，其左子树的值<根节点<右子树

思路：
若根节点>R,则考虑其左子树
若根节点<L,则考虑其右子树
以此类推

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {
    public TreeNode trimBST(TreeNode root, int L, int R) {

    if(root == null){
    return null;
}
if(root.val>R){
    return trimBST(root.left,L,R);
}
if(root.val<L){
    return trimBST(root.right,L,R);
}

root.left = trimBST(root.left,L,R);
root.right = trimBST(root.right,L,R);
return root;

    }
    }

leetcode_649

Posted on 2019-03-08

（java）
解法1：使用List+中序遍历
①中序遍历后，list里的值按从小到大排序
②从第一位i到最后一位j求sum开始遍历，若sum<k,则i++;sum>k,j–;sum==k,正确;遍历结束,错误。

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {
    public boolean findTarget(TreeNode root, int k){

    List<Integer> result = new ArrayList<Integer>();
inOrderTraversal(root,result);
for(int i = 0,j = result.size()-1;i<j;){
    if(result.get(i)+result.get(j)<k){
        i++;
    }
    else if(result.get(i)+result.get(j)>k){
        j--;
    }
    else{
        return true;
    }
}
return false;

    }

    private void inOrderTraversal(TreeNode node, List result) {

    if(node.left != null){
    inOrderTraversal(node.left,result);
}
result.add(node.val);
if(node.right != null){
    inOrderTraversal(node.right,result);
}

    }
    }

解法二：使用HashSet
①若节点root为空，则返回false；
②若set里含有值=（k-root.val），则返回true；
③否则，将该节点的值加入set；
遍历左右子树重复步骤①-③
代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public boolean findTarget(TreeNode root, int k) {
    Set<Integer> set = new HashSet();
return find(root, k, set);
    }
    public boolean find(TreeNode root, int k, Set set){
    if(root == null){
    return false;
}
if(set.contains(k-root.val)){
    return true;
}
set.add(root.val);
return find(root.left,k,set)||find(root.right,k,set);
    }
    }

Posted on 2019-03-06 | Edited on 2019-03-08

/**

  • Definition for a binary tree node.
  • struct TreeNode {
  • int val;
  • TreeNode *left;
  • TreeNode *right;
  • TreeNode(int x) : val(x), left(NULL), right(NULL) {}
  • };
    /
    class Solution {
    public:
    bool isSameTree(TreeNode     p, TreeNode* q) {
    //判断是否全为空
if (p == NULL && q == NULL){
    return true;
}
//判断是否一个空，一个非空
if (p == NULL || q == NULL){
    return false;
}
//两个非空，则判断该节点值是否相同
if(p->val == q->val){
    return isSameTree(p->left,q->left)&&isSameTree(p->right,q->right);
}
//若不等，则不相同
return false;
    }
    };
