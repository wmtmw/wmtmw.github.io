剑指面试题28-java-对称的二叉树

Posted on 2019-12-27

题目：判断二叉树是否是对称的

思路：
定义一种新的遍历方法，根节点-右节点-左节点
从同一个节点出发，同时进行前序遍历和新遍历

代码：
/*
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
boolean isSymmetrical(TreeNode pRoot)
{
if(pRoot == null){
return true;
}
return PanDuan(pRoot,pRoot);

}
boolean PanDuan(TreeNode root1,TreeNode root2){
    if(root1 == null && root2 == null){
        return true;
    }
    if((root1 != null && root2 == null)||(root1 == null && root2 != null)){
        return false;
    }
    if(root1.val != root2.val){
        return false;
    }
    return PanDuan(root1.left,root2.right)&&PanDuan(root1.right,root2.left);
}

}

剑指面试题27-java-二叉树的镜像

Posted on 2019-12-27

二叉树的镜像定义：源二叉树
8
/ \
6 10
/ \ / \
5 7 9 11
镜像二叉树
8
/ \
10 6
/ \ / \
11 9 7 5

思路：若当前节点不为空，则交换该节点的左右子节点
递归遍历左右节点。

代码：
/**
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
public void Mirror(TreeNode root) {
if(root!=null){
TreeNode temp;
temp = root.right;
root.right = root.left;
root.left = temp;
Mirror(root.left);
Mirror(root.right);
}
}
}

剑指面试题26-java-树的子结构

Posted on 2019-12-27

题目： 输入两棵二叉树A，B，判断B是不是A的子结构。（ps：我们约定空树不是任意一个树的子结构）

思路：在A中寻找一个节点Q，该节点和B的根节点R的值相等。
前序遍历A，直到找到该节点
如果根节点不等，则判断其左子节点是否相等
如果左子节点也不相等，则判断其右子节点是否相等
（这是一个递归的过程）

判断以Q为根节点的树是否包含和B相同的结构
若找到了该根节点，则遍历Q、R，判断其是否具有相同的结构
注意，空指针异常的边界条件，当R遍历到空指针，则说明该树枝判断完了，没有出现不等的情况，也就是该树枝是A的一部分。而当R中该树枝没有遍历完，而Q中已经没有节点了，说明结构不一致，可以提前结束，返回false。
若两树均没有出现null的情况，则继续递归判断，向递归函数中传入子节点，其在递归函数中按照新的根节点处理，如果当前节点相同，则继续递归判断其左右节点是否相同

代码：
/**
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
public boolean HasSubtree(TreeNode root1,TreeNode root2) {
if(root1 == null||root2==null){
return false;
}
boolean result = false;
if(root1.val == root2.val){
result = IfHas(root1,root2);
}
if(!result){
result = HasSubtree(root1.left,root2);
}
if(!result){
result = HasSubtree(root1.right,root2);
}
return result;
}
public boolean IfHas(TreeNode root1,TreeNode root2){
if(root2 == null){
return true;
}
if(root1 == null){
return false;
}
if(root1.val == root2.val){
return IfHas(root1.left,root2.left)&&IfHas(root1.right,root2.right);
}
return false;
}
}

剑指面试题25-java-合并两个排序的链表

Posted on 2019-12-26

题目： 输入两个单调递增的链表，输出两个链表合成后的链表，当然我们需要合成后的链表满足单调不减规则。

即输入两个链表：1->3->5,2->4->6 → 1->2->3->4->5->6

思路： 当其中一个链表为空时，返回另一个链表（其实就包括了两个链表均为空的情况）
两个链表都不为空时：
1.创建一个新的节点A，ListNode A = new ListNode(0);
2.比较两个链表头节点的值，选择值更小的节点，使节点A指向该节点，该节点所在链表的头节点改为该节点的下一个节点，循环操作，直到某个链表为空。
3.将新链表的尾节点指向非空链表的头节点。
4.返回A的下一个节点（特别注意，此时A为0，不是原始两个链表中的值，只是为了不需要区分对原始链表头节点和非头节点的操作而已）

代码：

/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}*/
public class Solution {
public ListNode Merge(ListNode list1,ListNode list2) {
//当链表1为空，则合并后的链表为链表2；
if(list1 == null){
return list2;
}
if(list2 == null){
return list1;
}
ListNode newNode = new ListNode(0);
ListNode curNode = newNode;
while(list1 != null && list2 != null){
if(list1.val < list2.val){
curNode.next = list1;
curNode = curNode.next;
list1 = list1.next;
}else{
curNode.next = list2;
curNode = curNode.next;
list2 = list2.next;
}
}
if(list1 == null){
curNode.next = list2;
}
if(list2 == null){
curNode.next = list1;
}
return newNode.next;
}
}

剑指面试题24-java-反转链表

Posted on 2019-12-23 | Edited on 2019-12-25

题目：输入一个链表，反转链表后，输出新链表的表头。

分析：三种情况
1.链表为空，返回空
2.链表中只有一个节点，返回该节点
3.链表中有多个节点

3的思路：
设置三个指针，前一个节点，当前节点，后一个节点
然后将当前节点指向前一个节点，遍历

代码
/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}*/
public class Solution {
public ListNode ReverseList(ListNode head) {
//如果链表为空
if(head == null){
return null;
}
//如果链表只有一个元素
if(head.next == null){
return head;
}
//如果链表有多个元素
//定义三个指针
ListNode pre = head;
ListNode cur = head.next;
ListNode next = null;

    while(cur!= null){
    //特别注意这个判断条件
        next = cur.next;
        cur.next = pre;
        pre = cur;
        cur = next;
    }
    head.next = null;
    return pre;

}

}

剑指面试题23-java-链表中环的入口节点

Posted on 2019-12-23

代码：
public class Solution {

public ListNode EntryNodeOfLoop(ListNode pHead)
{
    if(pHead==null||pHead.next==null){
        return null;
    }
    ListNode fast = pHead;
    ListNode slow = pHead;
    int stepCount = 0;
    while(fast.next!=null){
        fast = fast.next.next;
        slow = slow.next;
        stepCount++;
        if(fast == slow){
            slow = pHead;
            for(int i =0;i<stepCount;i++){
                if(fast == slow)
                {
                    return slow;
                }
                fast = fast.next;
                slow = slow.next;
            }
        }
    }
    return null;
}

}

剑指面试题22-java-链表中倒数第k个节点

Posted on 2019-12-22

题目：输入一个链表，输出该链表中倒数第k个结点。

复杂度分析：
时间复杂度：O(N);
空间复杂度：O(1).

特殊情况：1.链表为空；2.返回倒数第0个节点；3.链表不足k个节点

思路：
1.当指针为空，或者寻找倒数第0个节点，则返回null。
2.设置两个指针，第一个指针先走k-1步，如果链表节点不足，返回空；否则，当第一个指针走了k-1步后，两个指针同速度往前走，直到第一个指针到链表末尾。

代码：
/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}*/
public class Solution {
public ListNode FindKthToTail(ListNode head,int k) {
//判断：链表为空或者k<=0，均返回null
if(head==null||k<=0){
return null;
}
//创建两个指针fast，slow
ListNode fast = head, slow = head;
for(int i = 0;i < k-1;i++){
if(fast.next != null){
fast = fast.next;
}else{
return null;
}
}
while(fast.next!=null){
fast = fast.next;
slow = slow.next;
}
return slow;
}
}

简单写法：来源https://blog.csdn.net/littlehaes/article/details/91411250

public ListNode FindKthToTail(ListNode head,int k) {
ListNode p, q;
p = q = head;
int i = 0;
for (; p != null; i++) {
if (i >= k)
q = q.next;
p = p.next;
}
return i < k ? null : q;
}

剑指面试题21-java-调整数组顺序使奇数位于偶数前面

Posted on 2019-12-20

题目：输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

若不需要保证相对位置不变，则可以使用做法二，此时不需要开辟新空间

保证相对顺序的思路：
①创建一个新数组，数组大小与原数组相等。
②遍历原始数组，若为奇数，则保存在原始数组中；若为偶数，则保存在新数组中。
③遍历结束后，将新数组按顺序添加到原始数组奇数后

做法一：
public class Solution {
public void reOrderArray(int [] array) {
//如果数组为空或者数组元素小于2，则不需要重排
if(array == null || array.length<2){
return;
}

    int oddIndex = 0;
    int evenIndex = 0;
    int[] arrayEven = new int[array.length];
  //创建一个新数组，遍历原始数组，奇数按顺序保留在原始数组中，偶数按顺序添加到新数组中，遍历结束后，将新数组复制到原数组中
    for(int i = 0;i < array.length;i++){
        if((array[i]&1)!=0){
            array[oddIndex]=array[i];
            oddIndex++;
        }else{
            arrayEven[evenIndex]=array[i];
            evenIndex++;
        }
    }
    evenIndex=0;
    for(int i = oddIndex;i<array.length;i++){
        array[i]=arrayEven[evenIndex];
        evenIndex++;
    }
}

}

做法二：不保证相对顺序
public class Solution {
public void reOrderArray(int [] array) {
//如果数组为空或者数组元素小于2，则不需要重排
if(array == null || array.length<2){
return;
}
//设置两个索引，一个位于数组首位，一个位于数组末尾。
int oddIndex = 0;
int evenIndex = array.length-1;
while(oddIndex<evenIndex){
while(oddIndex<evenIndex&&(array[oddIndex]&1)!=0){
oddIndex++;
}
while(oddIndex<evenIndex&&(array[evenIndex]&1)==0){
evenIndex–;
}
if(oddIndex<evenIndex){
int temp = array[evenIndex];
array[evenIndex]=array[oddIndex];
array[oddIndex]=temp;
}
}
}
}

剑指面试题20-java-表示数值的字符串

Posted on 2019-12-19 | Edited on 2019-12-20

题目：请实现一个函数用来判断字符串是否表示数值（包括整数和小数）。例如，字符串”+100”,”5e2”,”-123”,”3.1416”和”-1E-16”都表示数值。 但是”12e”,”1a3.14”,”1.2.3”,”+-5”和”12e+4.3”都不是。

思路：
正确的形式有两种：+1.5,-1.5,1e-5,1e+5,-.123
也就是A.B, AEC,AeC的形式，A、C为正负整数，B为无符号整数
若为e指数，则A,C一定要正确
若为小数，则可以A存在B不存在，A不存在B存在（-.123），AB都存在，三种情况

1.判断A是否正确
2.判断是否是’.’,若是’.’,是则判断B是否为无符号整数（有一种错误情况是1.+1）
3.判断是否是’e’或’E’,是则判断C是否正确，C不正确则返回false
4.最终判断： ①index是否遍历完str，（若1.+1这种错误情况，index无法遍历完str）
②A,B都正确，或者A正确B不存在，或者A不存在B正确，即A||B

public class Solution {
//正确形式：+1.5
//+1e5
int index = 0;
public boolean isNumeric(char[] str) {
//判断字符串长度，若为null，则返回false
if(str == null){
return false;
}
//A为整数部分，B为小数部分，C为指数部分，A、C为正负整数，B为无符号整数
//先判断首位是否有符号，+/-，只能有一位
//判断整数部分，小数部分，指数部分是否是0~9
boolean A = isHaveFuhao(str);
boolean B = false,C = false;
//判断是小数形式（.）还是指数形式（e）
if(index<str.length && str[index]==’.’){
index++;
B = isShu(str);
}
if(index<str.length && (str[index]==’e’||str[index]==’E’)){
index++;
C = isHaveFuhao(str);
//如果出现了E，后面必须有整数，没有则返回false
if(C == false){
return false;
}
}
//index必须走完每个char，如果有A则为true，没有A则必须有B，才为true
return index == str.length && (A||B);
}
public boolean isHaveFuhao(char[] str){
if(index<str.length && (str[index]==’+’||str[index]==’-‘)){
index++;
}
return isShu(str);
}
public boolean isShu(char[] str){
int start = index;
while(index<str.length && str[index]>=’0’&&str[index]<=’9’){
index++;
}
return index>start;
}
}

剑指面试题19-java-正则表达式匹配

Posted on 2019-12-18

题目：
请实现一个函数用来匹配包括’.’和’‘的正则表达式。
模式中的字符’.’表示任意一个字符，而’‘表示它前面的字符可以出现任意次（包含0次）。
举例：字符串”aaa”与模式”a.a”和”abaca”匹配，但是与”aa.a”和”ab*a”均不匹配

思路：
1.模式第二个字符为’‘
若字符串不为空，则有三种匹配模式
① aaa与b，两者第一个字符不匹配，则字符串Index不动，模式Index后移两位，相当于x匹配零个字符
② aaa与a或者.：
两者第一个字符匹配，则字符串Index后移一位，模式Index后移两位
两者第一个字符匹配，字符串Index后移一位，模式Index不动，字符串的下一个字符继续匹配模式的第一个字符。
字符串为空，pattern不为空时，比如””和”x“/“xx“匹配
此时字符串Index不动，模式Index后移两位
2.模式第二个字符不为’*’
若字符串未匹配到串尾，若两者第一个字符匹配（两者相等，或者pattern第一位为’.’），则两者均后移一位继续匹配，否则，返回false

代码：
public class Solution {
public boolean match(char[] str, char[] pattern)
{
//1.如果两者都为空，则返回true
if(str == null && pattern == null){
return true;
}
//2.如果str不为空，pattern为空，则返回false
if(str != null && pattern == null){
return false;
}
//如果两者不为空，或者str为空，pattern不为空（比如pattern = b）
return Core(str,pattern,0,0);
}
public boolean Core(char[] str, char[] pattern,int strIndex,int patternIndex){
//结束条件：pattern和str都遍历完成
if(patternIndex == pattern.length && strIndex == str.length){
return true;
}
//如果pattern遍历完了，但str没有遍历完，则为false
if(patternIndex == pattern.length && strIndex != str.length){
return false;
}
//当模式中的第二个字符是“”
//字符串第1个和模式第1个匹配，则分3种匹配模式
if(patternIndex+1<pattern.length && pattern[patternIndex+1]==’‘){
if((strIndex!=str.length && pattern[patternIndex] == str[strIndex])||(strIndex!=str.length && pattern[patternIndex] == ‘.’)){
return Core(str,pattern,strIndex,patternIndex+2) //模式后移两位，相当于x匹配0个字符(aaa与b)
||Core(str,pattern,strIndex+1,patternIndex+2) //字符串后移一位，模式后移两位，相当于匹配一个字符（aaa与a或者.）
||Core(str,pattern,strIndex+1,patternIndex); //再匹配字符串中的下一个
}else{
return Core(str,pattern,strIndex,patternIndex+2); //当str为空，则strIndex == str.length,但与x仍然匹配！
}
}
//当模式中的第二个字符不是“*”，若两者第一个匹配，则都后移以为，否则返回false
if((strIndex != str.length && pattern[patternIndex]==str[strIndex])||(strIndex != str.length && pattern[patternIndex]==’.’)){
return Core(str,pattern,strIndex+1,patternIndex+1);
}
return false;
}
}
