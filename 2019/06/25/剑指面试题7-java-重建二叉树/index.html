剑指面试题7(java):重建二叉树

Posted on 2019-06-25

题目：
输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

分析：
前序遍历序列{1,2,4,7,3,5,6,8}
中序遍历序列{4,7,2,1,5,3,8,6}

1.根节点1，在中序序列中找到1，左边为左子树，右边为右子树
2.在前序序列中找到左子树的序号范围，在中序序列中找到左子树序号范围，递归
3.在前序序列中找到右子树的序号范围，在中序序列中找到右子树序号范围，递归

代码：
/**

  • Definition for binary tree
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    public class Solution {
    public TreeNode reConstructBinaryTree(int [] pre,int [] in) {

    //健壮性判断
if(pre.length!=in.length||pre.length <= 0||in.length <= 0){
    return null;
}
TreeNode node = reConstructBinaryTree(pre,0,pre.length-1,in,0,in.length-1);
return node;

    }
    public TreeNode reConstructBinaryTree(int [] pre,int preStart,int preEnd,int [] in,int inStart,int inEnd){

       //根节点
TreeNode root = new TreeNode(pre[preStart]); 
    //在中序遍历中找到根节点，左边为左子树，右边为右子树
int leftEnd = 0;
for(int i = inStart;i <= inEnd;i++){
    if(in[i] == pre[preStart]){
        break;
    }
    leftEnd++;
}
//左子树
if(leftEnd > 0){
    //在前序遍历中确定左子树元素的序号
    int preLeftStart = preStart+1;
    int preLeftEnd = preStart+leftEnd;
    //在中序遍历中确定左子树的序号
    int inLeftStart = inStart;
    int inLeftEnd = inStart+leftEnd-1;
    //递归
    root.left = reConstructBinaryTree(pre,preLeftStart,preLeftEnd,in,inLeftStart,inLeftEnd);
}
//右子树
if((preEnd-preStart)-leftEnd > 0){
    //在前序遍历中确定右子树元素的序号
    int preRightStart = preStart+leftEnd+1;
    int preRightEnd = preEnd;
    //在中序遍历中确定右子树的序号
    int inRigthStart = inStart+leftEnd+1;
    int inRightEnd = inEnd;
     //递归
    root.right = reConstructBinaryTree(pre,preRightStart,preRightEnd,in,inRigthStart,inRightEnd);

}
return root;

    }
    }
