剑指面试题5：替换空格

Posted on 2019-06-25

方法一：

分析：
①空格为一个char，替换成%20为三个char，会造成覆盖，所以不能直接替换。
②非常直观的想法就是，从后面往前替换，需要对原始的字符串缓冲区进行手动扩容。
③先一次遍历，获取字符串中空格的个数，扩容容量为2*空格个数。
④设两个指针，p1指向原始字符串末尾，p2指向扩容容量末尾；
若遇到空格，则依次替换为’0’,’2’.’%’,每次p2–，最后p1–；
否则，将p2处赋值为p1处的值；p2–，p1–；
直到p1到达字符串头部，结束。

public class Solution {
public String replaceSpace(StringBuffer str) {
//遍历字符串，找到字符串中空格的个数
int len = str.length();
int count = 0;
for(int i = 0;i < len;i++){
if(str.charAt(i)==’ ‘){
count += 2;
}
}
int p2 = len+count-1;
//手动扩容
for(int i = len;i<=p2;i++){
str.append(‘0’);
}
for(int p1 = len-1;p1>=0;p1–){
if(str.charAt(p1)==’ ‘){
str.setCharAt(p2,’0’);
p2–;
str.setCharAt(p2,’2’);
p2–;
str.setCharAt(p2,’%’);
p2–;
}else{
str.setCharAt(p2,str.charAt(p1));
p2–;
}
}
return str.toString();
}
}

方法二：

分析：
使用java中StringBuffer类中的insert()函数
从前往后遍历，遇到空格则删除空格，插入’%20’
具体实现实际是由类封装的函数实现的，并没有实现时间复杂度和空间复杂度的优化，并且不利于转换为其他语言。

public class Solution {
public String replaceSpace(StringBuffer str) {
//健壮性判断
if(str == null){
return null;
}
int len = str.length();
int i = 0;
while(i<len){
if(str.charAt(i)==’ ‘){
str.deleteCharAt(i);
str.insert(i,’%’);
i++;
str.insert(i,’2’);
i++;
str.insert(i,’0’);
len = len + 2;

        }
            i++;
    }
    return str.toString();
}

}
