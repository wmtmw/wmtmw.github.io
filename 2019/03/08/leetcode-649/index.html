leetcode_649

Posted on 2019-03-08

（java）
解法1：使用List+中序遍历
①中序遍历后，list里的值按从小到大排序
②从第一位i到最后一位j求sum开始遍历，若sum<k,则i++;sum>k,j–;sum==k,正确;遍历结束,错误。

代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    class Solution {
    public boolean findTarget(TreeNode root, int k){

    List<Integer> result = new ArrayList<Integer>();
inOrderTraversal(root,result);
for(int i = 0,j = result.size()-1;i<j;){
    if(result.get(i)+result.get(j)<k){
        i++;
    }
    else if(result.get(i)+result.get(j)>k){
        j--;
    }
    else{
        return true;
    }
}
return false;

    }

    private void inOrderTraversal(TreeNode node, List result) {

    if(node.left != null){
    inOrderTraversal(node.left,result);
}
result.add(node.val);
if(node.right != null){
    inOrderTraversal(node.right,result);
}

    }
    }

解法二：使用HashSet
①若节点root为空，则返回false；
②若set里含有值=（k-root.val），则返回true；
③否则，将该节点的值加入set；
遍历左右子树重复步骤①-③
代码：
/**

  • Definition for a binary tree node.
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }
  • }
    */
    class Solution {
    public boolean findTarget(TreeNode root, int k) {
    Set<Integer> set = new HashSet();
return find(root, k, set);
    }
    public boolean find(TreeNode root, int k, Set set){
    if(root == null){
    return false;
}
if(set.contains(k-root.val)){
    return true;
}
set.add(root.val);
return find(root.left,k,set)||find(root.right,k,set);
    }
    }
