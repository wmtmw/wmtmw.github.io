剑指面试题22-java-链表中倒数第k个节点

Posted on 2019-12-22

题目：输入一个链表，输出该链表中倒数第k个结点。

复杂度分析：
时间复杂度：O(N);
空间复杂度：O(1).

特殊情况：1.链表为空；2.返回倒数第0个节点；3.链表不足k个节点

思路：
1.当指针为空，或者寻找倒数第0个节点，则返回null。
2.设置两个指针，第一个指针先走k-1步，如果链表节点不足，返回空；否则，当第一个指针走了k-1步后，两个指针同速度往前走，直到第一个指针到链表末尾。

代码：
/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}*/
public class Solution {
public ListNode FindKthToTail(ListNode head,int k) {
//判断：链表为空或者k<=0，均返回null
if(head==null||k<=0){
return null;
}
//创建两个指针fast，slow
ListNode fast = head, slow = head;
for(int i = 0;i < k-1;i++){
if(fast.next != null){
fast = fast.next;
}else{
return null;
}
}
while(fast.next!=null){
fast = fast.next;
slow = slow.next;
}
return slow;
}
}

简单写法：来源https://blog.csdn.net/littlehaes/article/details/91411250

public ListNode FindKthToTail(ListNode head,int k) {
ListNode p, q;
p = q = head;
int i = 0;
for (; p != null; i++) {
if (i >= k)
q = q.next;
p = p.next;
}
return i < k ? null : q;
}
