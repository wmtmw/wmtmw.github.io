剑指面试题19-java-正则表达式匹配

Posted on 2019-12-18

题目：
请实现一个函数用来匹配包括’.’和’‘的正则表达式。
模式中的字符’.’表示任意一个字符，而’‘表示它前面的字符可以出现任意次（包含0次）。
举例：字符串”aaa”与模式”a.a”和”abaca”匹配，但是与”aa.a”和”ab*a”均不匹配

思路：
1.模式第二个字符为’‘
若字符串不为空，则有三种匹配模式
① aaa与b，两者第一个字符不匹配，则字符串Index不动，模式Index后移两位，相当于x匹配零个字符
② aaa与a或者.：
两者第一个字符匹配，则字符串Index后移一位，模式Index后移两位
两者第一个字符匹配，字符串Index后移一位，模式Index不动，字符串的下一个字符继续匹配模式的第一个字符。
字符串为空，pattern不为空时，比如””和”x“/“xx“匹配
此时字符串Index不动，模式Index后移两位
2.模式第二个字符不为’*’
若字符串未匹配到串尾，若两者第一个字符匹配（两者相等，或者pattern第一位为’.’），则两者均后移一位继续匹配，否则，返回false

代码：
public class Solution {
public boolean match(char[] str, char[] pattern)
{
//1.如果两者都为空，则返回true
if(str == null && pattern == null){
return true;
}
//2.如果str不为空，pattern为空，则返回false
if(str != null && pattern == null){
return false;
}
//如果两者不为空，或者str为空，pattern不为空（比如pattern = b）
return Core(str,pattern,0,0);
}
public boolean Core(char[] str, char[] pattern,int strIndex,int patternIndex){
//结束条件：pattern和str都遍历完成
if(patternIndex == pattern.length && strIndex == str.length){
return true;
}
//如果pattern遍历完了，但str没有遍历完，则为false
if(patternIndex == pattern.length && strIndex != str.length){
return false;
}
//当模式中的第二个字符是“”
//字符串第1个和模式第1个匹配，则分3种匹配模式
if(patternIndex+1<pattern.length && pattern[patternIndex+1]==’‘){
if((strIndex!=str.length && pattern[patternIndex] == str[strIndex])||(strIndex!=str.length && pattern[patternIndex] == ‘.’)){
return Core(str,pattern,strIndex,patternIndex+2) //模式后移两位，相当于x匹配0个字符(aaa与b)
||Core(str,pattern,strIndex+1,patternIndex+2) //字符串后移一位，模式后移两位，相当于匹配一个字符（aaa与a或者.）
||Core(str,pattern,strIndex+1,patternIndex); //再匹配字符串中的下一个
}else{
return Core(str,pattern,strIndex,patternIndex+2); //当str为空，则strIndex == str.length,但与x仍然匹配！
}
}
//当模式中的第二个字符不是“*”，若两者第一个匹配，则都后移以为，否则返回false
if((strIndex != str.length && pattern[patternIndex]==str[strIndex])||(strIndex != str.length && pattern[patternIndex]==’.’)){
return Core(str,pattern,strIndex+1,patternIndex+1);
}
return false;
}
}
