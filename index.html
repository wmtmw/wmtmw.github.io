剑指面试题21-java-调整数组顺序使奇数位于偶数前面

Posted on 2019-12-20

题目：输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

若不需要保证相对位置不变，则可以使用做法二，此时不需要开辟新空间

保证相对顺序的思路：
①创建一个新数组，数组大小与原数组相等。
②遍历原始数组，若为奇数，则保存在原始数组中；若为偶数，则保存在新数组中。
③遍历结束后，将新数组按顺序添加到原始数组奇数后

做法一：
public class Solution {
public void reOrderArray(int [] array) {
//如果数组为空或者数组元素小于2，则不需要重排
if(array == null || array.length<2){
return;
}

    int oddIndex = 0;
    int evenIndex = 0;
    int[] arrayEven = new int[array.length];
  //创建一个新数组，遍历原始数组，奇数按顺序保留在原始数组中，偶数按顺序添加到新数组中，遍历结束后，将新数组复制到原数组中
    for(int i = 0;i < array.length;i++){
        if((array[i]&1)!=0){
            array[oddIndex]=array[i];
            oddIndex++;
        }else{
            arrayEven[evenIndex]=array[i];
            evenIndex++;
        }
    }
    evenIndex=0;
    for(int i = oddIndex;i<array.length;i++){
        array[i]=arrayEven[evenIndex];
        evenIndex++;
    }
}

}

做法二：不保证相对顺序
public class Solution {

int oddIndex = 0;

while(oddIndex<evenIndex){
while(oddIndex<evenIndex&&(array[oddIndex]&1)!=0){
oddIndex++;
}
while(oddIndex<evenIndex&&(array[evenIndex]&1)==0){
evenIndex–;
}
if(oddIndex<evenIndex){
int temp = array[evenIndex];
array[evenIndex]=array[oddIndex];
array[oddIndex]=temp;
}
}
}

剑指面试题20-java-表示数值的字符串

Posted on 2019-12-19 | Edited on 2019-12-20

题目：请实现一个函数用来判断字符串是否表示数值（包括整数和小数）。例如，字符串”+100”,”5e2”,”-123”,”3.1416”和”-1E-16”都表示数值。 但是”12e”,”1a3.14”,”1.2.3”,”+-5”和”12e+4.3”都不是。

思路：
正确的形式有两种：+1.5,-1.5,1e-5,1e+5,-.123
也就是A.B, AEC,AeC的形式，A、C为正负整数，B为无符号整数
若为e指数，则A,C一定要正确
若为小数，则可以A存在B不存在，A不存在B存在（-.123），AB都存在，三种情况

1.判断A是否正确
2.判断是否是’.’,若是’.’,是则判断B是否为无符号整数（有一种错误情况是1.+1）
3.判断是否是’e’或’E’,是则判断C是否正确，C不正确则返回false
4.最终判断： ①index是否遍历完str，（若1.+1这种错误情况，index无法遍历完str）
②A,B都正确，或者A正确B不存在，或者A不存在B正确，即A||B

public class Solution {
//正确形式：+1.5
//+1e5
int index = 0;
public boolean isNumeric(char[] str) {
//判断字符串长度，若为null，则返回false
if(str == null){
return false;
}
//A为整数部分，B为小数部分，C为指数部分，A、C为正负整数，B为无符号整数
//先判断首位是否有符号，+/-，只能有一位
//判断整数部分，小数部分，指数部分是否是0~9
boolean A = isHaveFuhao(str);
boolean B = false,C = false;
//判断是小数形式（.）还是指数形式（e）
if(index<str.length && str[index]==’.’){
index++;
B = isShu(str);
}
if(index<str.length && (str[index]==’e’||str[index]==’E’)){
index++;
C = isHaveFuhao(str);
//如果出现了E，后面必须有整数，没有则返回false
if(C == false){
return false;
}
}
//index必须走完每个char，如果有A则为true，没有A则必须有B，才为true
return index == str.length && (A||B);
}
public boolean isHaveFuhao(char[] str){
if(index<str.length && (str[index]==’+’||str[index]==’-‘)){
index++;
}
return isShu(str);
}
public boolean isShu(char[] str){
int start = index;
while(index<str.length && str[index]>=’0’&&str[index]<=’9’){
index++;
}
return index>start;
}
}

剑指面试题19-java-正则表达式匹配

Posted on 2019-12-18

题目：
请实现一个函数用来匹配包括’.’和’‘的正则表达式。
模式中的字符’.’表示任意一个字符，而’‘表示它前面的字符可以出现任意次（包含0次）。
举例：字符串”aaa”与模式”a.a”和”abaca”匹配，但是与”aa.a”和”ab*a”均不匹配

思路：
1.模式第二个字符为’‘
若字符串不为空，则有三种匹配模式
① aaa与b，两者第一个字符不匹配，则字符串Index不动，模式Index后移两位，相当于x匹配零个字符
② aaa与a或者.：
两者第一个字符匹配，则字符串Index后移一位，模式Index后移两位
两者第一个字符匹配，字符串Index后移一位，模式Index不动，字符串的下一个字符继续匹配模式的第一个字符。
字符串为空，pattern不为空时，比如””和”x“/“xx“匹配
此时字符串Index不动，模式Index后移两位
2.模式第二个字符不为’*’
若字符串未匹配到串尾，若两者第一个字符匹配（两者相等，或者pattern第一位为’.’），则两者均后移一位继续匹配，否则，返回false

代码：
public class Solution {
public boolean match(char[] str, char[] pattern)
{
//1.如果两者都为空，则返回true
if(str == null && pattern == null){
return true;
}
//2.如果str不为空，pattern为空，则返回false
if(str != null && pattern == null){
return false;
}
//如果两者不为空，或者str为空，pattern不为空（比如pattern = b）
return Core(str,pattern,0,0);
}
public boolean Core(char[] str, char[] pattern,int strIndex,int patternIndex){
//结束条件：pattern和str都遍历完成
if(patternIndex == pattern.length && strIndex == str.length){
return true;
}
//如果pattern遍历完了，但str没有遍历完，则为false
if(patternIndex == pattern.length && strIndex != str.length){
return false;
}
//当模式中的第二个字符是“”
//字符串第1个和模式第1个匹配，则分3种匹配模式
if(patternIndex+1<pattern.length && pattern[patternIndex+1]==’‘){
if((strIndex!=str.length && pattern[patternIndex] == str[strIndex])||(strIndex!=str.length && pattern[patternIndex] == ‘.’)){
return Core(str,pattern,strIndex,patternIndex+2) //模式后移两位，相当于x匹配0个字符(aaa与b)
||Core(str,pattern,strIndex+1,patternIndex+2) //字符串后移一位，模式后移两位，相当于匹配一个字符（aaa与a或者.）
||Core(str,pattern,strIndex+1,patternIndex); //再匹配字符串中的下一个
}else{
return Core(str,pattern,strIndex,patternIndex+2); //当str为空，则strIndex == str.length,但与x仍然匹配！
}
}
//当模式中的第二个字符不是“*”，若两者第一个匹配，则都后移以为，否则返回false
if((strIndex != str.length && pattern[patternIndex]==str[strIndex])||(strIndex != str.length && pattern[patternIndex]==’.’)){
return Core(str,pattern,strIndex+1,patternIndex+1);
}
return false;
}
}

剑指面试题18-java-删除列表中重复节点

Posted on 2019-12-17

题目描述:
在一个排序的链表中，存在重复的结点，请删除该链表中重复的结点，重复的结点不保留，返回链表头指针。 例如，链表1->2->3->3->4->4->5 处理后为 1->2->5

思路：
1.如果头节点是重复节点，则从第一个不重复节点开始
2.如果头节点不是重复节点，则头节点指向下一个不重复的节点，返回头节点。

代码：
/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}
*/
public class Solution {
public ListNode deleteDuplication(ListNode pHead)
{
//健壮性判断：如果列表为空或者只有一个节点，则返回该列表
if(pHead == null||pHead.next == null){
return pHead;
}
ListNode right = pHead.next;
if(right!=null&&right.val == pHead.val){
//1.如果头节点是重复节点，则从第一个不重复节点开始
while(right!=null&&right.val == pHead.val){
right = right.next;
}
return deleteDuplication(right);
}else{
//2.如果头节点不是重复节点，则头节点指向下一个不重复的节点，返回头节点。
pHead.next = deleteDuplication(right);
return pHead;
}
}
}

        visited[row*cols+col] = true;
       // count = 1+movingCountCore(threshold,rows,cols,row-1,col,visited)+movingCountCore(threshold,rows,cols,row,col-1,visited)+movingCountCore(threshold,rows,cols,row+1,col,visited)+movingCountCore(threshold,rows,cols,row,col+1,visited);
        count++;
        movingCountCore(threshold,rows,cols,row-1,col,visited);
        movingCountCore(threshold,rows,cols,row,col-1,visited);
        movingCountCore(threshold,rows,cols,row+1,col,visited);
        movingCountCore(threshold,rows,cols,row,col+1,visited);
    }
    return count;
}
public boolean check(int threshold,int rows,int cols,int row,int col,boolean visited[]){
    if(row>=0&&row<rows&&col>=0&&col<cols&&getDigitSum(row)+getDigitSum(col)<=threshold&&!visited[row*cols+col]){
        return true;
    }
    return false;
}
public int getDigitSum(int num){
    int sum = 0;
    while(num > 0){
        sum += num %10;
        num/=10;
    }
    return sum;
}

}

剑指面试题12-java-矩阵中的路径

Posted on 2019-08-23

题目：给定一个矩阵，给定一组字符串，判断矩阵中是否存在一条包含字符串所有字符的路径。路径可以从矩阵中的任意一个格子开始，每一步可以再矩阵中向左向右向上向下移动，但不能两次经过格子。

思路：
①因为不能两次进入一个格子，所以需要一个和原始矩阵同维度的flag矩阵标识该格子是否可以进入。
boolean[] flag = new boolean[matrix.length];
for(int i = 0;i < matrix.length;i++){
flag[i] = true; //初始值全为true
}

②首先遍历矩阵，直到在矩阵中找到与字符串第一个值匹配的值，若找不到，则不存在路径满足。

③匹配到第i个字符所在的位置，寻找其上下左右，是否有一个位置的字符与第i+1个字符匹配

具体过程：
i）先匹配(上)：是否匹配，匹配，该位置置为false，重复③直到匹配完字符串中的所有字符；否则，该位置重新置为true，匹配(下);

ii）若匹配完上下左右，均无法匹配，则不存在一条路径满足要求，return false。

④注意越界判断，不可进入判断，不匹配判断。

我们可以看到i是一个自己调用自己的过程，独立使用一个函数，使用递归处理。

代码：
public class Solution {
public boolean hasPath(char[] matrix, int rows, int cols, char[] str)
{
//建立一个flag矩阵
boolean[] flag = new boolean[matrix.length];
for(int i = 0;i < matrix.length;i++){
flag[i] = true;
}
//寻找第一个匹配字符的位置
for(int i = 0;i < rows;i++){
for(int j = 0;j < cols;j++){
if(isKeXing(matrix, rows, cols,i,j,str,0,flag)){
return true;
}
}
}
return false;
}
//递归判断路径是否可行
public boolean isKeXing(char[] matrix, int rows, int cols,int i,int j, char[] str,int index,boolean[] flag){
//对应矩阵中的位置
int ind = i*cols+j;
//判断对应位置是否越界，是否与需要匹配的字符不相等，是否不能进入
if(i<0||i>=rows||j<0||j>=cols||matrix[ind]!=str[index]||flag[ind]==false){
return false;
}

//如果通过上述if，则将该位置置为false
flag[ind] = false;
//如果已经匹配完最后一个值了，则存在该路径
if(index == str.length-1){
    return true;
}
//递归判断是否存在该路径
if(isKeXing(matrix, rows, cols,i-1,j,str,index+1,flag)||isKeXing(matrix, rows, cols,i+1,j,str,index+1,flag)||isKeXing(matrix, rows, cols,i,j-1,str,index+1,flag)||isKeXing(matrix, rows, cols,i,j+1,str,index+1,flag)){
    return true;
}
//若上下左右都不可行
flag[ind] = true;
return false;

}
}

题目：现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。n<=39

递归很慢

解题思路：避免重复计算，把已经计算过的中间项保存下来。

具体代码：
public class Solution {
public int Fibonacci(int n) {
int f0 = 0;
int f1 = 1;
int f = 0;
if(n == 0){
return 0;
}else if(n == 1){
return 1;
}else{
for(int i = 2;i <=n;i++){
f = f0+f1;
f0 = f1;
f1 = f;
}
return f;
}

}

}

剑指面试题7(java):重建二叉树

Posted on 2019-06-25

题目：
输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

分析：
前序遍历序列{1,2,4,7,3,5,6,8}
中序遍历序列{4,7,2,1,5,3,8,6}

1.根节点1，在中序序列中找到1，左边为左子树，右边为右子树
2.在前序序列中找到左子树的序号范围，在中序序列中找到左子树序号范围，递归
3.在前序序列中找到右子树的序号范围，在中序序列中找到右子树序号范围，递归

代码：
/**

  • Definition for binary tree
  • public class TreeNode {
  • int val;
  • TreeNode left;
  • TreeNode right;
  • TreeNode(int x) { val = x; }

  • }
    */
    public class Solution {
    public TreeNode reConstructBinaryTree(int [] pre,int [] in) {

    //健壮性判断
if(pre.length!=in.length||pre.length <= 0||in.length <= 0){
    return null;
}
TreeNode node = reConstructBinaryTree(pre,0,pre.length-1,in,0,in.length-1);
return node;

    }
    public TreeNode reConstructBinaryTree(int [] pre,int preStart,int preEnd,int [] in,int inStart,int inEnd){

       //根节点
TreeNode root = new TreeNode(pre[preStart]); 
    //在中序遍历中找到根节点，左边为左子树，右边为右子树
int leftEnd = 0;
for(int i = inStart;i <= inEnd;i++){
    if(in[i] == pre[preStart]){
        break;
    }
    leftEnd++;
}
//左子树
if(leftEnd > 0){
    //在前序遍历中确定左子树元素的序号
    int preLeftStart = preStart+1;
    int preLeftEnd = preStart+leftEnd;
    //在中序遍历中确定左子树的序号
    int inLeftStart = inStart;
    int inLeftEnd = inStart+leftEnd-1;
    //递归
    root.left = reConstructBinaryTree(pre,preLeftStart,preLeftEnd,in,inLeftStart,inLeftEnd);
}
//右子树
if((preEnd-preStart)-leftEnd > 0){
    //在前序遍历中确定右子树元素的序号
    int preRightStart = preStart+leftEnd+1;
    int preRightEnd = preEnd;
    //在中序遍历中确定右子树的序号
    int inRigthStart = inStart+leftEnd+1;
    int inRightEnd = inEnd;
     //递归
    root.right = reConstructBinaryTree(pre,preRightStart,preRightEnd,in,inRigthStart,inRightEnd);

}
return root;

    }
    }

剑指面试题6(java)：从尾到头打印链表

Posted on 2019-06-25

题目描述：
输入一个链表，按链表值从尾到头的顺序返回一个ArrayList

分析：
①ArrayList是实现了基于动态数组的数据结构，LinkedList是基于链表的数据结构
②对于随机访问get和set，ArrayList优于LinkedList，因为LinkedList要移动指针
③对于新增和删除操作add和remove，LinkedList比较占优势，因为ArrayList要移动数据。

所以显然，遍历链表，使用LinkedList保存链表中的值（addFirst），保存完后不断remove，就是从尾到头的顺序。按照书本是打印，而这里是保存到ArrayList中。


/**
  • public class ListNode {
  • int val;
  • ListNode next = null;
    *
  • ListNode(int val) {
  • this.val = val;
  • }
  • }
    /
    import java.util.LinkedList;
    import java.util.ArrayList;
    public class Solution {
    public ArrayList printListFromTailToHead(ListNode listNode) {
    LinkedList ll = new LinkedList();
    while(listNode != null){
    ll.addFirst(listNode.val);
listNode = listNode.next;
    }
    ArrayList al = new ArrayList();
    while(!ll.isEmpty()){
    al.add(ll.remove());
    }
    return al;
    }
    }
