剑指面试题31-java-栈的压入,弹出序列

Posted on 2019-12-28

题目：输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否可能为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如序列1,2,3,4,5是某栈的压入顺序，序列4,5,3,2,1是该压栈序列对应的一个弹出序列，但4,3,5,1,2就不可能是该压栈序列的弹出序列。（注意：这两个序列的长度是相等的）

思路：假设入栈顺序是pushA[],出栈顺序是popA[]
建一个辅助栈s，依次将pushA压入s，每压入一个元素，循环判断栈顶元素是否和popA[]当前元素是否一致，一致则pop该栈顶元素，popA索引++，直到s的栈顶元素与popA[]元素不同，继续向s压入。直到pushA的元素压完，若popA索引长度和popA长度相同，则第二个序列为第一个序列的弹出顺序。

代码：
import java.util.Stack;

public class Solution {
public boolean IsPopOrder(int [] pushA,int [] popA) {
Stack s = new Stack<>();
int index = 0;
for(int i = 0;i < pushA.length;i++){
s.push(pushA[i]);
while(!s.isEmpty()&&(s.peek()==popA[index])){
s.pop();
index++;
}
}
return index == popA.length;
}

}

剑指面试题30-java-包含min函数的栈

Posted on 2019-12-27

题目：
定义栈的数据结构，请在该类型中实现一个能够得到栈中所含最小元素的min函数（时间复杂度应为O（1））

思路一：建一个辅助栈s2，若push的元素比s2的栈顶元素小，则压入新元素，否则将栈顶元素重新压栈

s2.peek():获取栈顶元素，但不弹出。

思路二：建一个辅助栈s2，若push的元素比s2的栈顶元素小，则压入新元素。
但pop的时候有变化。当s1要pop的元素与s2的栈顶元素相同时，s2才pop

代码一：
import java.util.Stack;

public class Solution {
Stack s1 = new Stack<>();
Stack s2 = new Stack<>();

public void push(int node) {
    s1.push(node);
    if(s2.isEmpty()){
        s2.push(node);
    }else{
        int temp = node<s2.peek()?node:s2.peek();
        s2.push(temp);
    }
}

public void pop() {
    s1.pop();
    s2.pop();
}

public int top() {
    return s1.peek();
}

public int min() {
    return s2.peek();
}

}

代码二：
import java.util.Stack;

public class Solution {
Stack s1 = new Stack<>();
Stack s2 = new Stack<>();

public void push(int node) {
    s1.push(node);
    if(s2.isEmpty()){
        s2.push(node);
    }else if(node<s2.peek()){
        s2.push(node);
    }
}

public void pop() {
    if (s1.pop()==s2.peek()){
        s2.pop();
    }
}

public int top() {
    return s1.peek();
}

public int min() {
    return s2.peek();
}

}

剑指面试题28-java-对称的二叉树

Posted on 2019-12-27

题目：判断二叉树是否是对称的

思路：
定义一种新的遍历方法，根节点-右节点-左节点
从同一个节点出发，同时进行前序遍历和新遍历

代码：
/*
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
boolean isSymmetrical(TreeNode pRoot)
{
if(pRoot == null){
return true;
}
return PanDuan(pRoot,pRoot);

}
boolean PanDuan(TreeNode root1,TreeNode root2){
    if(root1 == null && root2 == null){
        return true;
    }
    if((root1 != null && root2 == null)||(root1 == null && root2 != null)){
        return false;
    }
    if(root1.val != root2.val){
        return false;
    }
    return PanDuan(root1.left,root2.right)&&PanDuan(root1.right,root2.left);
}

}

剑指面试题27-java-二叉树的镜像

Posted on 2019-12-27

二叉树的镜像定义：源二叉树
8
/ \
6 10
/ \ / \
5 7 9 11
镜像二叉树
8
/ \
10 6
/ \ / \
11 9 7 5

思路：若当前节点不为空，则交换该节点的左右子节点
递归遍历左右节点。

代码：
/**
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
public void Mirror(TreeNode root) {
if(root!=null){
TreeNode temp;
temp = root.right;
root.right = root.left;
root.left = temp;
Mirror(root.left);
Mirror(root.right);
}
}
}

剑指面试题26-java-树的子结构

Posted on 2019-12-27

题目： 输入两棵二叉树A，B，判断B是不是A的子结构。（ps：我们约定空树不是任意一个树的子结构）

思路：在A中寻找一个节点Q，该节点和B的根节点R的值相等。
前序遍历A，直到找到该节点
如果根节点不等，则判断其左子节点是否相等
如果左子节点也不相等，则判断其右子节点是否相等
（这是一个递归的过程）

判断以Q为根节点的树是否包含和B相同的结构
若找到了该根节点，则遍历Q、R，判断其是否具有相同的结构
注意，空指针异常的边界条件，当R遍历到空指针，则说明该树枝判断完了，没有出现不等的情况，也就是该树枝是A的一部分。而当R中该树枝没有遍历完，而Q中已经没有节点了，说明结构不一致，可以提前结束，返回false。
若两树均没有出现null的情况，则继续递归判断，向递归函数中传入子节点，其在递归函数中按照新的根节点处理，如果当前节点相同，则继续递归判断其左右节点是否相同

代码：
/**
public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
public boolean HasSubtree(TreeNode root1,TreeNode root2) {
if(root1 == null||root2==null){
return false;
}
boolean result = false;
if(root1.val == root2.val){
result = IfHas(root1,root2);
}
if(!result){
result = HasSubtree(root1.left,root2);
}
if(!result){
result = HasSubtree(root1.right,root2);
}
return result;
}
public boolean IfHas(TreeNode root1,TreeNode root2){
if(root2 == null){
return true;
}
if(root1 == null){
return false;
}
if(root1.val == root2.val){
return IfHas(root1.left,root2.left)&&IfHas(root1.right,root2.right);
}
return false;
}
}

剑指面试题25-java-合并两个排序的链表

Posted on 2019-12-26

题目： 输入两个单调递增的链表，输出两个链表合成后的链表，当然我们需要合成后的链表满足单调不减规则。

即输入两个链表：1->3->5,2->4->6 → 1->2->3->4->5->6

思路： 当其中一个链表为空时，返回另一个链表（其实就包括了两个链表均为空的情况）
两个链表都不为空时：
1.创建一个新的节点A，ListNode A = new ListNode(0);
2.比较两个链表头节点的值，选择值更小的节点，使节点A指向该节点，该节点所在链表的头节点改为该节点的下一个节点，循环操作，直到某个链表为空。
3.将新链表的尾节点指向非空链表的头节点。
4.返回A的下一个节点（特别注意，此时A为0，不是原始两个链表中的值，只是为了不需要区分对原始链表头节点和非头节点的操作而已）

代码：

/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}*/
public class Solution {
public ListNode Merge(ListNode list1,ListNode list2) {
//当链表1为空，则合并后的链表为链表2；
if(list1 == null){
return list2;
}
if(list2 == null){
return list1;
}
ListNode newNode = new ListNode(0);
ListNode curNode = newNode;
while(list1 != null && list2 != null){
if(list1.val < list2.val){
curNode.next = list1;
curNode = curNode.next;
list1 = list1.next;
}else{
curNode.next = list2;
curNode = curNode.next;
list2 = list2.next;
}
}
if(list1 == null){
curNode.next = list2;
}
if(list2 == null){
curNode.next = list1;
}
return newNode.next;
}
}

剑指面试题24-java-反转链表

Posted on 2019-12-23 | Edited on 2019-12-25

题目：输入一个链表，反转链表后，输出新链表的表头。

分析：三种情况
1.链表为空，返回空
2.链表中只有一个节点，返回该节点
3.链表中有多个节点

3的思路：
设置三个指针，前一个节点，当前节点，后一个节点
然后将当前节点指向前一个节点，遍历

代码
/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}*/
public class Solution {
public ListNode ReverseList(ListNode head) {
//如果链表为空
if(head == null){
return null;
}
//如果链表只有一个元素
if(head.next == null){
return head;
}
//如果链表有多个元素
//定义三个指针
ListNode pre = head;
ListNode cur = head.next;
ListNode next = null;

    while(cur!= null){
    //特别注意这个判断条件
        next = cur.next;
        cur.next = pre;
        pre = cur;
        cur = next;
    }
    head.next = null;
    return pre;

}

}

剑指面试题23-java-链表中环的入口节点

Posted on 2019-12-23

代码：
public class Solution {

public ListNode EntryNodeOfLoop(ListNode pHead)
{
    if(pHead==null||pHead.next==null){
        return null;
    }
    ListNode fast = pHead;
    ListNode slow = pHead;
    int stepCount = 0;
    while(fast.next!=null){
        fast = fast.next.next;
        slow = slow.next;
        stepCount++;
        if(fast == slow){
            slow = pHead;
            for(int i =0;i<stepCount;i++){
                if(fast == slow)
                {
                    return slow;
                }
                fast = fast.next;
                slow = slow.next;
            }
        }
    }
    return null;
}

}

剑指面试题22-java-链表中倒数第k个节点

Posted on 2019-12-22

题目：输入一个链表，输出该链表中倒数第k个结点。

复杂度分析：
时间复杂度：O(N);
空间复杂度：O(1).

特殊情况：1.链表为空；2.返回倒数第0个节点；3.链表不足k个节点

思路：
1.当指针为空，或者寻找倒数第0个节点，则返回null。
2.设置两个指针，第一个指针先走k-1步，如果链表节点不足，返回空；否则，当第一个指针走了k-1步后，两个指针同速度往前走，直到第一个指针到链表末尾。

代码：
/*
public class ListNode {
int val;
ListNode next = null;

ListNode(int val) {
    this.val = val;
}

}*/
public class Solution {
public ListNode FindKthToTail(ListNode head,int k) {
//判断：链表为空或者k<=0，均返回null
if(head==null||k<=0){
return null;
}
//创建两个指针fast，slow
ListNode fast = head, slow = head;
for(int i = 0;i < k-1;i++){
if(fast.next != null){
fast = fast.next;
}else{
return null;
}
}
while(fast.next!=null){
fast = fast.next;
slow = slow.next;
}
return slow;
}
}

简单写法：来源https://blog.csdn.net/littlehaes/article/details/91411250

public ListNode FindKthToTail(ListNode head,int k) {
ListNode p, q;
p = q = head;
int i = 0;
for (; p != null; i++) {
if (i >= k)
q = q.next;
p = p.next;
}
return i < k ? null : q;
}
